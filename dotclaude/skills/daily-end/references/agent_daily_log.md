# エージェント日次ログの確認（毎日・必須）

今日のログ・セッション内容から、面談・ゼミ・相談を行ったエージェントを特定する。
該当するエージェントごとに `.claude/agent-memory/[agent-id]/daily/YYYY-MM-DD.md` が存在するか確認する。

- **存在しない場合**: セッション内容の要約を作成し、`.claude/agent-memory/[agent-id]/daily/YYYY-MM-DD.md` に Write ツールで保存する。ディレクトリがなければ作成する。
- **存在する場合**: 内容を Read で確認し、今日の追加セッションがあれば追記する。

結果を以下の形式でログに記録する：

```markdown
### 📝 エージェント日次ログ確認
- ✅ [agent-id]: daily/YYYY-MM-DD.md 保存済み
- ⚠️ [agent-id]: 面談記録なし → 作成・保存しました
```

> **重要**: この記録がなければハートビート（金曜自動・随時手動）でメモリが更新されない。面談内容が失われる前に必ず記録すること。
