# 監査レポートテンプレート

以下の形式で結果をユーザーに提示する。

```
# リポジトリ監査レポート — YYYY-MM-DD

## 検出された問題

### 🔴 要対処（旧形式の残骸・壊れた参照）
| # | 問題 | 対象ファイル | 提案 |
|---|------|-------------|------|
| 1 | ... | ... | 削除 / アーカイブ / 更新 |

### 🟡 推奨（改善の余地あり）
| # | 問題 | 対象ファイル | 提案 |
|---|------|-------------|------|

### ✅ 正常
[問題なしの項目を簡潔にリスト]

## 提案アクション一覧
1. [具体的なアクション]
2. ...
```
