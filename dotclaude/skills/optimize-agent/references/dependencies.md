# 依存スキルと後処理

## 依存スキル一覧

エージェント定義を変更すると、以下のスキルの参照先に影響が出る可能性がある。
**optimize-agent では更新しない**（別途対応）。最適化完了時に注意喚起のみ行う。

| スキル | 確認すべき点 |
|--------|-------------|
| `consult-agent` | エージェントIDの変更に追従しているか |
| `repo-audit` | エージェント一覧のチェックロジックが正しいか |

## メモリの扱い

`.claude/agent-memory/[エージェントID]/MEMORY.md` が存在する場合：
- エージェントIDを変更した場合はメモリディレクトリ名も更新する
- 不要になった旧メモリはユーザーに確認してから削除する
