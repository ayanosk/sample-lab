# 最適化案テンプレート

ユーザーへの承認依頼時に以下の形式で提示する。

```
【最適化案: [エージェントID]】

■ 現行（.claude/agents/[部門]/[id].md）: [行数]行
■ 最適化後: 見込み[行数]行

フロントマター:
  name: [id（ハイフン区切り）]
  description: [1文の説明]
  tools: [推奨ツール]
  model: [推奨モデル or 省略]

system_prompt 変更点:
  - [削除] CLAUDE.md と重複する行動規約（xx行）
  - [削除] 具体的ファイルパス（xx行）
  - [圧縮] 専門分野リスト（xx行 → xx行）
  - [保持] [保持する内容の要約]
```
