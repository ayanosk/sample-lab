# AGENTS.md テンプレート

Claude Code、Codex など複数の AI ツールで共有する基本ルールです。最初に自分用へ書き換えて使います。

## このリポジトリの目的

- `[例: 研究メモと分析コードを一元管理する]`
- `[例: 発表資料と論文下書きを整理する]`
- `[例: AI と安全に共同作業する]`

## 応答

- `[応答言語]` で応答する
- 作業前に、何をするかを短く伝える
- 作業後に、何ができたかを報告する

## 作業の進め方

- 対象プロジェクトの `README.md` を先に読む
- `sources/` は入力、`outputs/` は出力として扱う
- 分析や整形は `scripts/` を優先して再利用する
- 根拠やメモは `docs/` に残す

## 安全

- 機密情報は追加しない
- 大きな生成物や一時ファイルは `.gitignore` で除外する
- 破壊的な操作は、必要性を説明してから行う

## 研究室ごとの追加ルール

- `[例: 原稿の外部公開前に人間が確認する]`
- `[例: 数値や固有名詞は元資料に当たって再確認する]`
- `[例: 毎日の作業ログを docs/ に残す]`

## ツール構成

このリポジトリは Claude Code 向けに最適化されています。
エージェント定義は `.claude/agents/`、スキルは `.claude/skills/` に配置されています。

Codex を中心に使う場合は、「Codex 中心に使えるように最適化して」と指示してください。
`.claude/agents/` と `.claude/skills/` を `.agents/` にコピーし、
AGENTS.md に必要な情報を集約する作業を自動で行います。
