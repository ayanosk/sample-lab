\maketitle

\section{目的}

このファイルは、LaTeX 環境が正しく動作しているかを確認するためのサンプルです。
以下の項目が正常に表示されていれば、環境構築は成功しています。

\begin{enumerate}
  \item 日本語の表示（明朝体・ゴシック体）
  \item 欧文の表示（Times 系セリフ体）
  \item 参考文献の引用と一覧表示
  \item 脚注と相互参照
\end{enumerate}

\section{日欧混在テスト}

日本語と英語を混在させた文章です。
Claude Code や Codex は AI エージェントであり、ターミナルや IDE から作業を補助できます。
フランス語の引用も確認します：\textit{L'Académie royale des sciences} は1666年に設立されました。

ゴシック体のテスト：{\sffamily この部分はゴシック体（サンセリフ）で表示されます。}

\section{参考文献の引用テスト}


日本語文献の引用テスト\autocite[10--15]{yamada2020}。
欧文文献の引用テスト\autocite[42]{smith2019}。
同じ文献の再引用（短縮形になるか確認）\autocite[20]{yamada2020}。

\section{脚注テスト}

本文中の脚注テストです\footnote{これは手動の脚注です。参考文献の自動脚注（autocite）とは別に動作します。}。
